# Debianmaker

**Debianmaker** is a lightweight tool designed to build customized, 100% reproducible Debian Live ISOs. It enables a **Stateless Desktop** workflow by decoupling the operating system from user data.

## Key Features
* **Immutable by design:** Generate clean images that run entirely in RAM (`toram`), ensuring a fresh system state at every boot.
* **Ventoy Optimized:** Purpose-built to be deployed via Ventoy. Updating your system is as simple as copying a new ISO file to your USB drive.
* **Persistent Home:** Designed to work with a dedicated `/home` partition. By labeling a partition as `persistence` (and including a `persistence.conf` file), you can keep your personal data and settings across reboots while the OS remains immutable.
* **Image-based Updates:** Replace the entire OS by swapping a single file—eliminating the risk of broken dependencies during system upgrades.

## Quick Setup

### 1. Prepare the Persistence Partition
To keep your data across reboots while keeping the OS immutable, prepare a partition on your internal SSD:

1.  **Label the partition**: Set the filesystem label to `persistence`.
2.  **Configure persistence**: Mount the partition and create a file named `persistence.conf`:
    ```bash
    echo "/home" > /mnt/persistence.conf
    ```
    *(Note: This ensures only your `/home` directory stays persistent).*

### 2. Deploy with Ventoy and `toram`
1.  Copy the ISO generated by **Debianmaker** to your **Ventoy** USB drive.
2.  Boot from the USB and ensure the boot parameters include `persistence` and `toram`.

#### Why use `toram`?
* **Incredible Speed:** The entire Operating System is loaded into RAM. System calls and application launches are near-instant, limited only by your memory bandwidth.
* **USB Longevity:** Once the boot process is finished, the USB drive is no longer accessed by the OS, preventing wear and tear and allowing for safe removal if needed.
* **Full Immutability:** Any change made to the system files during the session exists only in volatile memory and is wiped clean upon reboot.

### 3. Updating the System
Updating is atomic and risk-free:
1.  Generate a new ISO with **Debianmaker**.
2.  Mount the Ventoy USB and replace the old `.iso` with the new one.
3.  **Reboot.** Your OS is now updated, and your `/home` on the internal SSD remains untouched and ready to use.
